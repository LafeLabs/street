Trash Magic Servers

Trash Magic servers are computers we remove all personal and private information from, put a free operating system on, put a web server on, and then load up with self-replicating media using the system of Trash Magic self-replicating code.

To make a Trash Magic Server, follow the instructions on one of the Github repositories or read the local readme on this server. Servers are not private property. They are outside of the property system, they represent a digital commons which the Trash Magicians will use to spread the memes of Trash Magic along with all the memes of the networks we support.
